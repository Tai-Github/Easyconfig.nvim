## II. Installation

### NOTE: Make sure you have to install `nodejs`, `npm` and `yarn`.

```bash
sudo apt install nodejs npm && sudo npm install -g yarn
```

### 1. Clone nvim setup:

```bash
git clone https://github.com/Tai-Github/package.nvim.git ~/.config/nvim
```
